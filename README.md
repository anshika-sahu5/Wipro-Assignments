The assignment folder is organized as follows:

*   **`Topic-01-ER-Modeling/`**: Conceptual database design, Entity-Relationship diagrams, and schema design.
*   **`Topic-02-Relational-Calculus/`**: Theoretical database queries, relational algebra, and tuple relational calculus.

*   **`Topic-05-PL-SQL/`**: Advanced database programming including Triggers, Stored Procedures, and Cursors (if applicable).
